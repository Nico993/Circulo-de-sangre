POBRETITO

Sistema para registrar y gestionar reclamos de los ciudadanos de la ciudad de San Francisco.

#####Ejecutar en local:

#Requsistos previos:
    Instalar nodejs
    Instalar mongodb

#Ejecutar client:
    cd Client/pobre-tito
    npm install
    npm start

#Ejecutar server:
    cd Server
    npm install
    node server.js
